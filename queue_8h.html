◆ queue_create()

t_queue* queue_create ( void  )

Crea una cola.

Devuelve
Retorna un puntero a la cola creada, liberable con:
  • queue_destroy() si se quiere liberar la cola pero no los elementos que contiene.
  • queue_destroy_and_destroy_elements() si se quiere liberar la cola con los elementos que contiene.
